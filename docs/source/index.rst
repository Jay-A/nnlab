Overview
--------

``nnlab`` is an experimental neural network framework designed
to explore the relationship between:

- model architecture,
- activation behavior,
- optimization,
- feature representation,
- learning outcomes.


The goal is not to replace production machine learning frameworks,
but to provide a computational laboratory for understanding how
learning systems are constructed and how individual components
influence behavior.
